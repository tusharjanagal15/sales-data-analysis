# Sales Data Analysis Project

## Objective
Analyze sales data to identify trends and improve business decisions.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Monthly Sales Trends
- City-wise Sales Analysis
- Hourly Sales Patterns

## Key Insights
- December has highest sales
- San Francisco is top-performing city
- Peak sales occur in evening hours

## Dataset
Source: Kaggle
